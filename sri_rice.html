Type: Redesigning SRI-RICE's website platform and visual appearance.
Role: Visual design and input. I was responsible for the majority of the UI decisions and actively worked with the communications department every design step.
Timeline: October 2017 - Present

Overview: SRI-RICE is a agricultural sustainability website that contains the most comprehensive collection of information on the System of Rice Intensification globally. Many of the documents published come directly from the partners: Farmers, researchers, NGOs, government agencies and other stakeholders from around the world. The company has been around for more than 30 years and its website was outdated and need of a visual redesign. In addition, the old website could only be run by one developer, making it difficult to add information to the website. My task was to redesign the website using Drupal and add features that would allow others to add content easily.

Problem

What was the need to redesign the SRI-RICE website.....

Though the old website was thorough and sufficient for the company, it lacked easy collaboration between developers to edit and add information. Furthermore, it was outdated and lacked modern design. The old website used Dreamweaver and though it was easy to use, only one person could edit at a time, making research addition to the website difficult especially if the main administrator was not available.

Solution

The need to switch to Drupal was necessary because not only could multiple people edit and be administrators at the same time, but also administrators did not need to be able to code in order to do so. Creating templates for each set of research allowed administrators to easily add information. Making the website more modern would help bring in more visitors as well as help differentiate the company from other competitors.
